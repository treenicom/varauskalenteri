# VARAUSKALENTERI

This project started as an exercise for a course at Haaga-Helia UAS (https://terokarvinen.com/python-weppipalvelu-ideasta-tuotantoon/). It is my goal to get it into production in the Spring of 2026.

## What is it?

Varauskalenteri or Reservation Calendar is aimed at personal trainers / fitness instructors to handle reservations for their services. It requires no signing up from the customer.
